##### Abstract
This paper brieviy describes the Autonomous Architecture Over Restricted Domains project (A2RD). In It begins with the description of the context upon which the project is focused, and in the sequence describes the A2RD abstract and implementation models. Finish by presenting the environment conceptual model named Structure for Knowledge Acquisition, Use and Collaboration Inter A2RD Agents (SKAU), showing where stand the components, inputs and facilities required to interact among the intelligent agents of the various implementations in their respective and restricted routing domains (Autonomous Systems) that build the Internet.
